# `abs`

The `abs` filter returns the absolute value of a number.

## Syntax

```template
abs(number_value)
```

## Parameters

- `number_value` (number): The input number.

## Example

```template
abs(-42)
```

**Output:**

```
42
```

## Source Code

```tsx
export const abs = (input: number): number => {
  return Math.abs(input);
};
```
