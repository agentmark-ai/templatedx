# Syntax Highlighting

Since TemplateDX is a subset of MDXjs, it works well out of the box for any MDX-based syntax highlighters.

Download for VSCode: https://marketplace.visualstudio.com/items?itemName=unifiedjs.vscode-mdx

IntelliJ: https://plugins.jetbrains.com/plugin/14944-mdx

Eclipse: https://open-vsx.org/extension/unifiedjs/vscode-mdx
